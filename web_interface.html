Setup Instructions

Computer Setup
  1. Install Python 3.7+ on your computer
  2. Install PyBluez: pip install pybluez
  3. Run the service: python computer_service.py
  4. Ensure Bluetooth is enabled and discoverable
Android App Setup
  1. Install the Android app on your device
  2. Enable Bluetooth and grant permissions
  3. Scan for devices and select your computer
  4. Connect and start controlling your PC
